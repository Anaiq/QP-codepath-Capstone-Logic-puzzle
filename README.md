### Overview
Description
The "Logic Puzzle" app is an interactive iOS game developed with Swift to challenge problem-solving skills. It features a user-friendly interface, easy puzzle, hints, and answer checks. The project demonstrates my skills in Swift, UIKit for the interface, and Core Data to store user progress and puzzle settings.

### Product Spec
1. User Stories:   
Logic Puzzles is a small mobile software company in Portland that delivers unique and fun logic puzzles on the go. 
My ideal customer is a woman named Meena. Meena is in their mid 30’s/40's. She has a masterss degree and manages a mid-size Human Resources team in Portland. She’s a natural problem solver and loves solving problems.She travels a lot for work.  Her go to travel companions are logic problem books but can sometimes be cumbersome to carry. She wants something fun to do while waiting during travels.
